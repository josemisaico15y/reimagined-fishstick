# Bot de Telegram para Render

## Instrucciones
1. Sube este proyecto a un repositorio en GitHub.
2. Ve a https://render.com y crea un nuevo 'Web Service' usando este repositorio.
3. En la sección 'Environment', agrega una variable de entorno:
   - Key: BOT_TOKEN
   - Value: TU_TOKEN_DE_BOTFATHER
4. En 'Start Command', coloca:
   python bot.py
5. Haz Deploy y el bot estará activo 24/7.
